## 使用方式

- 安装依赖 ：`npm install`
- 执行所有测试 ：`npm run test`
- 运行指定单个测试方式 ：`./node_modules/jest/bin/jest.js ./src/data-structures/linked-list/__test__/LinkedListNode.test.js`
- 代码 Lint 校验：`npm run lint`
- 配置了 husky `pre-commit` hooks，提交代码之前校验代码和执行单元测试
